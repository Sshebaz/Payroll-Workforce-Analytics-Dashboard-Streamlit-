# Payroll & Workforce Analytics Dashboard

An interactive data analytics dashboard built with Streamlit to analyze employee payroll, attendance, and departmental performance.

## Key Features

* Payroll dataset creation using real-world structured data
* Salary insights (total & average compensation)
* Department-wise salary analysis
* Attendance trend analysis
* Interactive data visualizations (bar & line charts)

## Tech Stack

* Python
* Pandas
* Matplotlib
* Streamlit

## Use Case

This project demonstrates how organizations can monitor workforce costs and productivity trends through simple analytics dashboards.

## 🔗 Live Demo: https://5zlv9y5t4trf9b6j8d2cxt.streamlit.app/
